# ht16k33

Rust driver for the Holtek HT16K33 "RAM Mapping 16*8 LED Controller Driver with keyscan"

You can integrate `ht16k33` into your project through the [releases on crates.io](https://crates.io/crates/ht16k33):

```toml
# Cargo.toml
[dependencies]
ht16k33 = "0.1.0"
```
